安 装 指 南 
 
1. 首先上传所有文件到服务器

2. 安装此系统请执行admin/install.htm,程序会指导您的安装！ 

3. 如果你是 Unix 类服务器，那么你还要做个额外的操作，设置include/config.inc.php文件777。 
如果你使用的是 Windows 类服务器，那么无需做任何权限的设置。 

4.在程序根目录下建立cache,upload文件夹，属性改为777

最后重复提示一次： 检查服务器上的 admin目录 中是不是还存在 install.php 如果有，请务必删除，否则非常危险，切记切记！
